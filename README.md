## Setup

### 1. Clone the repository

```sh
git clone https://github.com/Aspyse/HalftoneDemo.git
cd HalftoneDemo
```

### 2. Configure

```sh
cmake --preset x64-release
```

### 3. Build

```sh
cmake --build out\build\x64-release
```

### 4. Run executable

```sh
.\out\build\x64-release\HalftoneDemo.exe
```
